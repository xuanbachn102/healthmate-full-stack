========================================
GOOGLE OAUTH - HƯỚNG DẪN NHANH
========================================

🔗 LINK: https://console.cloud.google.com/

BƯỚC 1: TẠO PROJECT
--------------------
1. Click "Select a project" (góc trên trái)
2. Click "NEW PROJECT"
3. Tên: HealthMate
4. Click CREATE
5. Chọn project vừa tạo

BƯỚC 2: ENABLE API
------------------
1. Menu trái → APIs & Services → Library
2. Search: "People API"
3. Click "Google People API"
4. Click "ENABLE"

BƯỚC 3: OAUTH CONSENT
---------------------
1. Menu trái → APIs & Services → OAuth consent screen
2. Chọn "External" → CREATE
3. Điền:
   - App name: HealthMate
   - User support email: email của bạn
   - Developer email: email của bạn
4. SAVE AND CONTINUE
5. ADD OR REMOVE SCOPES:
   - Tick: userinfo.email
   - Tick: userinfo.profile
   - Tick: openid
   - UPDATE → SAVE AND CONTINUE
6. ADD USERS: Thêm email của bạn
7. SAVE AND CONTINUE
8. BACK TO DASHBOARD

BƯỚC 4: TẠO CLIENT ID
----------------------
1. Menu trái → APIs & Services → Credentials
2. + CREATE CREDENTIALS
3. Chọn "OAuth client ID"
4. Application type: "Web application"
5. Name: HealthMate Web Client
6. Authorized JavaScript origins:
   + ADD URI: http://localhost:5173
   (KHÔNG có dấu / cuối)
7. CREATE
8. COPY "Your Client ID"
   Dạng: xxxxx-xxxxx.apps.googleusercontent.com
   LƯU LẠI!

BƯỚC 5: CẤU HÌNH .ENV
---------------------
1. Mở file: frontend/.env
2. Thêm dòng này:

VITE_GOOGLE_CLIENT_ID=paste-client-id-ở-đây.apps.googleusercontent.com

   (Thay paste-client-id-ở-đây bằng Client ID thật)
   (KHÔNG có dấu ngoặc kép)

3. Lưu file

BƯỚC 6: CHẠY VÀ TEST
--------------------
Terminal 1:
cd backend
npm start

Terminal 2:
cd frontend
npm run dev

Trình duyệt:
http://localhost:5173/login

Phải thấy:
✅ Nút "Sign in with Google"
✅ Click → Popup Google mở
✅ Chọn tài khoản → Đăng nhập thành công

========================================
LỖI THƯỜNG GẶP
========================================

❌ "Access blocked"
→ Chưa setup OAuth consent screen
→ Quay lại BƯỚC 3

❌ "redirect_uri_mismatch"
→ Check Authorized JavaScript origins
→ Phải có: http://localhost:5173 (không dấu /)

❌ Không thấy nút Google
→ Check frontend/.env có VITE_GOOGLE_CLIENT_ID
→ Restart frontend

❌ "Invalid Client ID"
→ Check Client ID trong .env có đúng
→ Không có space, không có ngoặc kép

========================================
CHECK NHANH
========================================

✓ Google Cloud Project đã tạo?
✓ People API đã enable?
✓ OAuth consent screen đã setup?
✓ Client ID đã tạo?
✓ Client ID đã copy?
✓ Đã thêm vào frontend/.env?
✓ Backend đang chạy port 4000?
✓ Frontend đang chạy port 5173?

NẾU TẤT CẢ ✓ → NÚT GOOGLE SẼ HOẠT ĐỘNG!
